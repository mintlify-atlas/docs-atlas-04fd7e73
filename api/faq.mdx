## FAQ Categories

Manage frequently asked question categories organized by user role.

### Create FAQ Category

<ParamField path="POST" type="endpoint">
  `/api/faq_categories`
</ParamField>

Creates a new FAQ category.

**Request Body**

<ParamField body="name" type="string" required>
  Name of the FAQ category
</ParamField>

<ParamField body="roleScope" type="enum" required>
  Target user role. Values: `GUIDE`, `TOURIST`, `BOTH`
</ParamField>

<ParamField body="sortOrder" type="integer">
  Display order for the category (lower numbers appear first)
</ParamField>

**Response**

<ResponseField name="faqCategoryId" type="integer">
  Unique identifier for the FAQ category
</ResponseField>

<ResponseField name="name" type="string">
  Category name
</ResponseField>

<ResponseField name="roleScope" type="enum">
  Target user role: `GUIDE`, `TOURIST`, or `BOTH`
</ResponseField>

<ResponseField name="sortOrder" type="integer">
  Display order
</ResponseField>

**Example Request**

```bash
curl -X POST http://localhost:8080/api/faq_categories \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Reservaciones y Pagos",
    "roleScope": "TOURIST",
    "sortOrder": 1
  }'
```

**Example Response**

```json
{
  "faqCategoryId": 5,
  "name": "Reservaciones y Pagos",
  "roleScope": "TOURIST",
  "sortOrder": 1
}
```

## FAQ Items

Manage individual FAQ questions and answers.

### Create FAQ Item

<ParamField path="POST" type="endpoint">
  `/api/faq_items`
</ParamField>

Creates a new FAQ item (question and answer pair).

**Request Body**

<ParamField body="faqCategoryId" type="integer" required>
  ID of the parent FAQ category
</ParamField>

<ParamField body="question" type="string" required>
  The FAQ question
</ParamField>

<ParamField body="answer" type="string" required>
  The answer to the question
</ParamField>

<ParamField body="isActive" type="boolean">
  Whether this FAQ item is active/visible. Default: `true`
</ParamField>

<ParamField body="sortOrder" type="integer">
  Display order within the category (lower numbers appear first)
</ParamField>

**Response**

<ResponseField name="faqItemId" type="integer">
  Unique identifier for the FAQ item
</ResponseField>

<ResponseField name="faqCategoryId" type="integer">
  Parent category ID
</ResponseField>

<ResponseField name="question" type="string">
  The question
</ResponseField>

<ResponseField name="answer" type="string">
  The answer
</ResponseField>

<ResponseField name="isActive" type="boolean">
  Active status
</ResponseField>

<ResponseField name="sortOrder" type="integer">
  Display order
</ResponseField>

**Example Request**

```bash
curl -X POST http://localhost:8080/api/faq_items \
  -H "Content-Type: application/json" \
  -d '{
    "faqCategoryId": 5,
    "question": "¿Cómo puedo cancelar mi reservación?",
    "answer": "Puedes cancelar tu reservación desde tu panel de control. Las políticas de cancelación varían según el tour y el guía. Revisa los términos específicos en tu confirmación de reserva.",
    "isActive": true,
    "sortOrder": 1
  }'
```

**Example Response**

```json
{
  "faqItemId": 142,
  "faqCategoryId": 5,
  "question": "¿Cómo puedo cancelar mi reservación?",
  "answer": "Puedes cancelar tu reservación desde tu panel de control. Las políticas de cancelación varían según el tour y el guía. Revisa los términos específicos en tu confirmación de reserva.",
  "isActive": true,
  "sortOrder": 1
}
```
