## Newsletter Subscriptions

Manage email newsletter subscriptions.

### Create Newsletter Subscription

<ParamField path="POST" type="endpoint">
  `/api/newsletter_subscriptions`
</ParamField>

Creates a new newsletter subscription.

**Request Body**

<ParamField body="email" type="string" required>
  Email address to subscribe
</ParamField>

<ParamField body="sourcePage" type="string">
  Page URL where the subscription was initiated
</ParamField>

<ParamField body="isActive" type="boolean">
  Whether the subscription is active. Default: `true`
</ParamField>

<ParamField body="createdAt" type="string">
  Timestamp when subscription was created (ISO 8601 format)
</ParamField>

<ParamField body="unsubscribedAt" type="string">
  Timestamp when user unsubscribed (ISO 8601 format). `null` if still subscribed
</ParamField>

**Response**

<ResponseField name="subscriptionId" type="integer">
  Unique identifier for the subscription
</ResponseField>

<ResponseField name="email" type="string">
  Subscribed email address
</ResponseField>

<ResponseField name="sourcePage" type="string">
  Source page URL
</ResponseField>

<ResponseField name="isActive" type="boolean">
  Active status
</ResponseField>

<ResponseField name="createdAt" type="string">
  Creation timestamp
</ResponseField>

<ResponseField name="unsubscribedAt" type="string">
  Unsubscribe timestamp (null if active)
</ResponseField>

**Example Request**

```bash
curl -X POST http://localhost:8080/api/newsletter_subscriptions \
  -H "Content-Type: application/json" \
  -d '{
    "email": "laura.garcia@example.com",
    "sourcePage": "https://kinconecta.com/",
    "isActive": true
  }'
```

**Example Response**

```json
{
  "subscriptionId": 5623,
  "email": "laura.garcia@example.com",
  "sourcePage": "https://kinconecta.com/",
  "isActive": true,
  "createdAt": "2026-03-11T15:45:12",
  "unsubscribedAt": null
}
```

## Common Use Cases

### Subscribe to Newsletter

To subscribe a user to the newsletter:

```bash
curl -X POST http://localhost:8080/api/newsletter_subscriptions \
  -H "Content-Type: application/json" \
  -d '{
    "email": "newuser@example.com",
    "isActive": true
  }'
```

### Unsubscribe from Newsletter

To unsubscribe a user (set `isActive` to `false` and record timestamp):

```bash
curl -X PUT http://localhost:8080/api/newsletter_subscriptions/5623 \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "isActive": false,
    "unsubscribedAt": "2026-03-11T16:30:00"
  }'
```

### Re-subscribe

To re-subscribe a previously unsubscribed user:

```bash
curl -X PUT http://localhost:8080/api/newsletter_subscriptions/5623 \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "isActive": true,
    "unsubscribedAt": null
  }'
```
