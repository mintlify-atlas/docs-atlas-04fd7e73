# Quickstart Guide

Get up and running with Kin Conecta quickly. This guide will help you create your account and complete your first meaningful action on the platform.

<Note>
  Choose your path: Are you a **tourist** looking for authentic experiences, or a **local guide** ready to share your expertise?
</Note>

## For Tourists: Find Your First Guide

Follow these steps to create a tourist account and discover your first guide.

<Steps>
  <Step title="Create Your Account">
    Navigate to the registration page and select **Turista** (Tourist) as your role.
    
    Fill in your information:
    ```html
    - Full name (nombre completo)
    - Date of birth (fecha de nacimiento)
    - Country code (clave LADA)
    - Phone number (teléfono)
    - Email address (correo electrónico)
    - Password (contraseña)
    ```
    
    <Warning>
      You must be 18 years or older to create an account.
    </Warning>
  </Step>
  
  <Step title="Complete Your Profile">
    After registration, enhance your profile by adding:
    - **Interests**: Select categories like gastronomy, adventure, history, or culture
    - **Languages**: Add languages you speak
    - **Profile photo**: Upload a profile picture
    - **Travel preferences**: Set your travel style and preferences
    
    This helps the platform recommend the best guides for you.
  </Step>
  
  <Step title="Explore Destinations">
    Use the explore page to discover guides and tours:
    
    1. Navigate to the **Explorar** section from your dashboard
    2. Use the search bar to find destinations or activities:
    ```javascript
    // Search by destination, guide name, or activity type
    "Oaxaca gastronomy"
    "Yucatan adventure"
    "Mexico City history tour"
    ```
    3. Filter results by category:
       - Gastronomía
       - Aventura
       - Historia
       - Cultura
  </Step>
  
  <Step title="Find and Contact a Guide">
    Browse guide profiles and tour offerings:
    
    1. Click on any tour card to view details
    2. Review the guide's:
       - Experience and certifications
       - Average rating and reviews
       - Available tour plans
       - Pricing and duration
    3. Add guides to your favorites for later
    4. Contact the guide directly through the platform
    
    <Note>
      Look for the **Guía verificado** (Verified Guide) badge for added trust.
    </Note>
  </Step>
  
  <Step title="Book Your First Experience">
    When you've found the perfect tour:
    
    1. Select your preferred date from the guide's calendar
    2. Review the tour details and what's included
    3. Submit your booking request
    4. Wait for guide confirmation
    5. Receive booking confirmation and trip details
    
    You can track all your bookings in the **Trips** section of your dashboard.
  </Step>
</Steps>

## For Guides: Create Your First Tour

Follow these steps to set up your guide account and publish your first tour offering.

<Steps>
  <Step title="Register as a Guide">
    Go to the registration page and select **Guía** (Guide) as your role.
    
    Complete the registration form with:
    ```html
    - Full name
    - Date of birth
    - Country code and phone number
    - Email address
    - Password
    ```
    
    <Warning>
      Guides must be at least 18 years old and may need to complete verification steps.
    </Warning>
  </Step>
  
  <Step title="Build Your Guide Profile">
    Create a compelling profile that attracts tourists:
    
    **Essential Information:**
    - Profile photo and cover image
    - Bio and description of your expertise
    - Location(s) where you offer services
    - Languages you speak
    
    **Showcase Your Expertise:**
    - Add certifications and credentials
    - List your areas of expertise (history, gastronomy, adventure, etc.)
    - Set your availability and preferences
    - Add adaptation offerings (accessibility options)
    
    <CodeGroup>
    ```json API Example - Guide Profile
    {
      "userId": 1,
      "bio": "Experienced local guide specializing in cultural tours",
      "yearsOfExperience": 5,
      "hourlyRate": 500.00,
      "maxGroupSize": 10,
      "isVerified": true,
      "verificationDate": "2026-03-01"
    }
    ```
    
    ```json API Example - Expertise Areas
    {
      "guideUserId": 1,
      "expertiseAreaId": 2,
      "proficiencyLevel": "EXPERT",
      "yearsOfExperience": 5
    }
    ```
    </CodeGroup>
  </Step>
  
  <Step title="Create Your First Tour Plan">
    Navigate to your guide dashboard and create a tour:
    
    **Tour Details:**
    1. Give your tour a compelling title
    2. Write a detailed description
    3. Select tour category (gastronomy, adventure, culture, etc.)
    4. Set duration and pricing
    5. Define minimum and maximum group sizes
    
    **Add Destinations:**
    - List the places you'll visit
    - Add descriptions for each stop
    - Set the order of destinations
    
    **Included Items:**
    - Transportation
    - Meals and beverages
    - Entry tickets
    - Equipment or materials
    - Insurance coverage
    
    <CodeGroup>
    ```json API Example - Create Tour
    {
      "guideUserId": 1,
      "categoryId": 1,
      "title": "Authentic Oaxaca Food Tour",
      "description": "Experience traditional Oaxacan cuisine",
      "duration": 4,
      "durationUnit": "HOURS",
      "basePrice": 850.00,
      "currency": "MXN",
      "maxGroupSize": 8,
      "minGroupSize": 2,
      "isActive": true
    }
    ```
    
    ```json API Example - Add Included Item
    {
      "tourId": 1,
      "itemType": "MEAL",
      "itemDescription": "Traditional Oaxacan mole tasting",
      "isIncluded": true
    }
    ```
    </CodeGroup>
  </Step>
  
  <Step title="Set Your Availability">
    Manage your calendar to show when you're available:
    
    1. Go to the **Calendar** section
    2. Mark your available dates and times
    3. Block dates when you're unavailable
    4. Set recurring availability patterns
    
    Tourists can only book times when you're marked as available.
  </Step>
  
  <Step title="Publish and Promote">
    Final steps to go live:
    
    1. Review all your tour details
    2. Ensure your profile is complete
    3. Set your tour status to **Active**
    4. Your tour is now visible to tourists!
    
    <Note>
      Monitor your dashboard for booking requests and messages from potential clients.
    </Note>
  </Step>
</Steps>
