# Welcome to Kin Conecta

**Kin Conecta** is a digital travel social network platform designed to connect national and international tourists with **local guides** in Mexico.

<Note>
  **Kin** comes from Maya and means "Sun", representing connection, energy, and human closeness.
</Note>

## What is Kin Conecta?

Kin Conecta addresses the problem where travelers often encounter generic and standardized experiences. Our solution enables travel personalization by allowing users to directly contact local experts to live authentic and secure experiences.

The platform facilitates meaningful connections between tourists seeking unique experiences and local guides who know their destinations intimately.

## Core Features

### Role-Based Access
Kin Conecta supports two primary user roles:
- **Tourist**: Search, discover, and book experiences
- **Guide**: Create offerings and manage tour services

### Intelligent Search
Filter destinations by city/state and categories of interest including:
- Gastronomy
- Adventure
- History
- Culture
- And more

### Tour Management
Guides can create, edit, and manage comprehensive tour itineraries with:
- Destination details
- Included items and services
- Pricing and duration
- Availability calendar

### Security & Verification
Built-in user validation and authentication system to ensure safe connections between tourists and guides.

### Compatibility Profiling

<Note>
Kin Conecta features a unique compatibility algorithm based on research about matching experience expectations, rhythm, interests, and logistics between tourists and guides.
</Note>

The profiling system analyzes compatibility to:
- Generate personalized recommendations
- Optimize travel experiences
- Increase satisfaction and safety for both profiles

## Architecture

The platform uses a **Monolithic (MVC)** architecture evolving towards REST services, with:
- RESTful API endpoints for all operations
- Spring Data JPA for database access
- Component-based frontend architecture
- Responsive design for mobile and desktop

## API-First Design

Kin Conecta exposes a comprehensive REST API with 195+ endpoints across 39 controllers, covering:
- User management and authentication
- Tourist and guide profiles
- Tour and destination management
- Bookings and trip management
- Reviews and ratings
- Messaging and notifications
- FAQ and support systems
