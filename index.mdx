## Conectando experiencias, iluminando destinos

Kin Conecta is a travel social network platform that connects tourists with verified local guides in Mexico. Our platform enables authentic, personalized travel experiences through direct connections between travelers and local experts.

**Kin** (Maya for "Sun") represents connection, energy, and human closeness.

## Platform Architecture

Kin Conecta is built with modern technologies for performance and scalability:

- **Backend**: Spring Boot 4.0, Java 21, MySQL 8
- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap 5
- **Architecture**: REST API with 195+ endpoints across 39 controllers
